# Svelte + Vite

Este projeto utiliza Svelte com Vite para criar uma experiência interativa para festas PopCorn.

## Como rodar

```bash
pnpm install
pnpm run dev
```

## Build para produção

```bash
pnpm run build
```

## Deploy

O deploy pode ser feito facilmente na Vercel, sem necessidade de configuração especial. O diretório `dist` será servido como estático.
